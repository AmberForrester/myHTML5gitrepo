HTML Colgroup

Style specific columns of a table

Only a limited selection of CSS properties allowed to be used in the colgroup.

The colgroup tag must be a child of a table element and should be placed before any other table elements, like thead, tr, td, etc.

Place the colgroup tag after the caption element, if present.


Legal CSS Properties

  1. width
  2. visibility
  3. background
  4. border

HTML Table Colgroup

Style the first two columns of a table, using the colgroup + col elements.

The colgroup element should be used as a container for the column specifications.

Each group is specified with a col element.
