HTML Block and Inline Elements


Block-level Elements

  1. Always starts on a new line
  2. Always take up the full width available
  3. Browser automatically add some space (margin) before and after the element

Two commonly used block elements:

The p element - defines a paragraph in an HTML document.
The div element - defines a section in an HTML document.


The div Element

  1. Control certain tags within the div tag
  2. Used as a container for other HTML elements
  3. By default a block element = takes up all available width, and comes with line breaks before and after
  4. 3 Common Attributes Used: style, class, id

Inline Elements

  1. Only takes up as much width as necessary
  2. Control certain content on the page

Using the span element to control text within a paragraph:

The SPAN element is an inline element, and will not start on a new line and only takes up as much width as necessary.


The span Element

  1. Inline container used to mark up a part of a text, or a part of a document
  2. When used together with CSS, it can be used to style parts of the text
  3. 3 Common Attributes Used: style, class, id

HTML Class Attribute

  1. Used to specify a class for any HTML element
  2. Case Sensitive
  3. Multiple HTML elements can share the same class
  4. Specify groups of similar styles
  5. Different elements can share the same class

Point to a class name in a style sheet. Also used by JavaScript to access and manipulate elements with the specific class name.

HTML id Attribute

  1. Case Sensitive
  2. Used to specify a unique id for an HTML element
  3. The value must be unique within the HTML document
  4. Used to point to a specific style declaration in a style sheet

The syntax for id includes:
- write a hash character (#)
- followed by an id name
- define the CSS properties within the curly braces {}
